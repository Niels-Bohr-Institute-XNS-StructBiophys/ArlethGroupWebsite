Calculating Pair distance distribution functions for Proteins

This open source program (C and Python) calculates the pair distance distribution function (PDDF) p(r) from a high-resolution protein structure in PDB format, and the scattering intensity can be calculated by a Fourier transform of the PDDF. A water layer can be added as an option.

This software is developed, and maintained, by Andreas Haahr Larsen.
The code repository of CaPP is hosted on the GitHub page of CaPP.
