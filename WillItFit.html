Will It Fit

WillItFit is an analysis tool for refining structural models from small-angle scattering data. It relies on standard minimisation methods to find models of maximum likelihood from (several sets of) data. The code is open source and written with a focus on flexibility and customisability. In particular, models written in C can be added to the library using the outlined formalisms.

The inner workings of the software are described in:
Pedersen, Arleth, & Mortensen (2013)
J. Appl. Crystallogr. 46, 1894-1898


The code repositories of WillItFit are currently hosted on Sourceforge
